bootstrap 学习

组织排版内容，更多的请参考官方文档。

1、 内容一般都包含在class为：container-fluid 的div里面

2、图片自适应 class=”img-reponsive”

3、文字居中 class=”text-center” 另外文字 text-danger 文字红色

4、 按钮 class=”btn” 或者是 ntn-block btn-primary btn-info btn-danger注意他们的区别

5、 网格布局 全局被 div.row 包裹 每个块被div.col-xs- 或者div.col-md- 包裹

6、 Font Awesome 图标库非常好用 i元素 指代图标 i.fa fa-thumbs-up 等等

7、 class=”well” Bootstrap 有一个 class 属性叫做 well，它的作用是为设定的列创造出一种视觉上的深度感。
