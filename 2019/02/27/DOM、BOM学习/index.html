事件

如何绑定事件处理函数：

ele.onxxx = function(){}

兼容性很好，但是一个元素的同一个事件只能绑定一个处理程序

基本等同于写在html行间上

  1. obj.addEventListen(type,fn,false) //IE9一下不兼容 一个事件可以绑定多个处理程序。
  1. obj.attachEvent(‘on’ + type + fn) //IE 独有 一个事件可以绑定多个处理程序。

封装兼容性很好的绑定函数

事件解除

事件冒泡

结构上嵌套的关系元素，会存在事件冒泡的功能，即同一事件，自子元素冒泡向父元素。（自底向上）

事件捕获

结构上嵌套关系的元素，会存在事件捕获的功能，即同一事件，自父元素捕获至子元素（自顶向下）

IE没有捕获事件

触发顺序： 先捕获，后冒泡

focus blur change submit reset select 等事件不冒泡。

阻止冒泡：

div.onclick = function(e){
e.stopPropagation(); //不支持IE9一下的版本
}

阻止默认事件

默认事件： 表单提交，a标签跳转，右键菜单。

  1. return false 以对象属性的方式注册的事件才生效。

  2. event.preventDefault();

  3. event.returnValue = false; //兼容IE
